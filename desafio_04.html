DESAFÍO 4 EN LÓGICA

DE PROGRAMACIÓN DE ONE

Los resultados de los ejercicios de esta parte de los desafíos se mostrarán por consola. Presionar F12 para poder ver los resultados en consola.

Funciones

Ejercico 1: Crear Lista

1. Crea una lista vacía llamada "listaGenerica".

Ejercicio 2: Crear Lista de Lenguajes de Programación

2. Crea una lista de lenguajes de programación llamada "lenguagesDeProgramacion" con los siguientes elementos:
'JavaScript', 'C', 'C++', 'Kotlin' y 'Python'.

Ejercio 3: Agregar Elementos a la Lista

3. Agrega a la lista "lenguagesDeProgramacion los siguientes elementos: 'Java', 'Ruby' y 'GoLang'.

Función 4: Función Para Mostrar los Elementos de la Lista

4. Crea una función que muestre en la consola todos los elementos de la lista "lenguagesDeProgramacion".

    Función 5: Función Para Mostrar los Elementos de la Lista de Forma Inversa

    5. Crea una función que muestre en la consola todos los elementos de la lista "lenguagesDeProgramacion" en orden inverso.

      Función 6: Promedio de Lista de Números

      6. Crea una función que calcule el promedio de los elementos en una lista de números.

      Función 7: Muestra Números Extremos

      7. Crea una función que muestre en la consola el número más grande y el número más pequeño en una lista.

      Función 8: Suma de sus Elementos

      8. Crea una función que devuelva la suma de todos los elementos en una lista.

      Función 9: Posición de su Elemento

      9. Crea una función que devuelva la posición en la lista donde se encuentra un elemento pasado como parámetro,
      o -1 si no existe en la lista.

      Función 10: Lista Resultante

      10. Crea una función que reciba dos listas de números del mismo tamaño y devuelva una nueva lista con la suma de los
      elementos uno a uno.

        Función 11: Lista de Cuadrados

        11. Crea una función que reciba una lista de números y devuelva una nueva lista con el cuadrado de cada número.
